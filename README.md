# Full Stack E-Commerce Platform

## Project Overview

Developed a full-stack e-commerce application that enables users to browse products, search and filter inventory, view product details, manage a shopping cart, and place orders through a responsive web interface. The application integrates a React frontend with a Node.js and Express backend, using RESTful APIs for communication and MongoDB for data persistence.

## Key Features

* Product catalog with dynamic product listing
* Product search and filtering functionality
* Product detail pages with individual item information
* Shopping cart management using React Context API
* Order creation and management workflows
* RESTful API integration between frontend and backend
* MongoDB-based data storage and retrieval
* Pagination for scalable product browsing
* Automated unit and integration testing
